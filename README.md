# AmmoQuick

Developed with Unreal Engine 4.24
Some enhancements on the Sample FPS Project provided by Unreal Engine

Available Features:
	Ammo and Clip System
	Ammo auto Pickups
	Reload(R)
	Shooting Recoil
	Explosive Shots
	Double Jump
	Warp to Focused Area(Left Alt)
	Sprint(Left Shift + Movement)
